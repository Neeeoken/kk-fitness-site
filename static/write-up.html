Write-Up: KK Fitness Website

Theme and Purpose

We created this website inspired by my partner, Neoken Ocampo, who has now become a certified gym rat. Our website is a multi-page static site designed to promote a fitness center. Its main goal is to provide users with information about the services we offer and the fitness products available. The target audience includes fitness enthusiasts, potential gym members, and individuals looking for fitness-related products and services. The website aims to deliver a clean, user-friendly, and responsive experience across all devices.

HTML Structure

The website is built using semantic HTML5 elements to ensure proper structure and accessibility:

  • Header: Contains the logo and navigation bar for easy access to all pages.
  • Navigation Bar: Links to internal pages, including Home, Services, About, and Contact.
  • Main Content: Each page uses <section> and <article> elements to organize content logically.
  • Footer: Includes copyright information and additional navigation links.

Key features include:

  • A form on the Contact page for user inquiries.
  • A table on product pages (e.g., Clothes, Supplements, Accessories) to display pricing and product details.
  • Embedded media elements, such as images, to enhance visual appeal.

CSS Styling

The website uses an external CSS file (styles.css) for consistent styling across all pages. Key design choices include:

  • Color Scheme: A combination of dark and light tones to create a professional and modern look.
  • Typography: Clean, sans-serif fonts for readability.
  • Layout: Flexbox is used for layout design, ensuring proper alignment and spacing of elements.
  • Hover Effects: Interactive hover effects are applied to navigation links and buttons to improve user engagement.

Responsiveness

The website is fully responsive, ensuring compatibility with various devices (desktop, tablet, and mobile). Media queries are used to adjust layouts and font sizes for smaller screens. For example:

  • The navigation bar adapts to smaller screens by stacking links vertically.
  • The features section switches to a single-column layout on mobile devices.

Design Choices

The design choices were made to balance aesthetics and functionality:

  • Hero Section: A visually appealing hero section on the homepage highlights the fitness center's mission.
  • Product Pages: Tables are used to organize product details for clarity.
  • Contact Form: A simple and accessible form allows users to reach out easily.

Conclusion

Our website demonstrates the application of fundamental web development concepts, including semantic HTML, CSS styling, and responsive design. It is a user-friendly, visually appealing, and functional website that meets the project requirements.
